# Forecasting Human Development and Regional Inequality in India

## Overview
Analysis of HDI trends across Indian states (1990–2022) using time series forecasting and inequality metrics.

## Key Tasks
- Forecasted HDI trends till 2030 using ARIMA, SARIMA, and Exponential Smoothing
- Performed time-series decomposition, missing value handling, and regional disparity analysis
- Analysed regional inequality using Gini coefficients (CV)
- Identified development trajectory variations across states
- Built an interactive Tableau dashboard for HDI trends, state rankings, and regional convergence

## Dataset
- Source:  Global Data Lab
- Coverage: 30+ Indian states and UTs, 1990–2022

## Tools & Libraries
Python, Pandas, Statsmodels, Matplotlib, Seaborn, Tableau

## Key Findings
- Gini coefficient (CV) declined from ~0.10 (1990) to ~0.045 (2022) — states are converging
- Goa ranked highest HDI (0.76), Bihar lowest (0.57) in 2022
